# 课程管理系统介绍 java69

## 项目概述

课程管理系统是一个基于前后端分离架构的系统，旨在为教务管理人员、教师及学生提供一个高效、便捷的在线服务平台。

## 技术栈

- 后端：SpringBoot、Mybatis-Plus
- 前端：Vue
- 数据库：MySQL

## 功能模块

### 系统角色

系统包含以下三种角色：

- 教务管理员
- 教师
- 学生

### 功能介绍

#### 教务管理员

- 管理员首页
- 系管理
- 专业管理
- 班级管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

#### 教师

- 教师首页
- 授课查询
- 教师课表
- 成绩录入

#### 学生

- 学生首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护

## 运行环境

- 后端运行环境：JDK 1.8 或更高版本，兼容 SpringBoot 的应用服务器
- 前端运行环境：现代浏览器
- 数据库环境：MySQL 5.7 或更高版本

## 部署步骤

1. 准备后端运行环境
2. 准备数据库环境并导入数据库脚本
3. 部署前端代码至 Web 服务器
4. 启动后端服务
5. 通过前端访问后端服务

## 目录结构

```
- backend                # 后端代码目录
  - src
    - main
      - java
      - resources
- frontend               # 前端代码目录
  - src
- db_script              # 数据库脚本目录
```

## 核心亮点

- 基于成熟的 SpringBoot + Vue 技术栈，易于维护和扩展
- 角色权限分明，满足不同用户的需求
- 功能全面，涵盖教务管理的各个方面
- 前后端分离，易于部署和开发协作
- 代码结构清晰，方便二次开发
